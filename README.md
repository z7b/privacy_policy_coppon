## تفعيل GitHub Pages (مهم — بدونه يظهر 404)

1. افتح: https://github.com/z7b/privacy_policy_coppon/settings/pages
2. **Build and deployment → Source:** اختر **GitHub Actions**
3. بعد أول push، سيعمل workflow `Deploy GitHub Pages` تلقائيًا
4. أو شغّله يدويًا: **Actions** → **Deploy GitHub Pages** → **Run workflow**
5. انتظر 1–3 دقائق ثم افتح الرابط أعلاه
